📱 Transfer to New Phone

Moving phones? Use one of these methods to bring all your data across.

Option A — Google Sheet Sync
If you have a Sheet connected below, just open this app on the new phone, connect the same Sheet URL, and tap ⬇ Pull All — all your boxes and entries will appear.
Option B — Export / Import JSON
No Sheet? Export your data as a file, share it to your new phone, then import it.

☁️ Connect Google Sheet

One sheet holds data for all your boxes. Each row is tagged with the box name so you can filter in the sheet.

  1. Go to drive.google.com → create a new Google Sheet.
  2. Click Extensions → Apps Script.
  3. Delete the default code, paste in the google-apps-script-v2.js file contents.
  4. Click Save, then Deploy → New deployment.
  5. Type = Web App · Execute as = Me · Access = Anyone → Deploy.
  6. Copy the Web App URL and paste it below.
